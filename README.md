# Binary-Parser
Linux Binary Parser



# Instructions
      ./compile
     
 View ELf Header
      
      ./parser <binary> header
     
 View Elf Segments(Program header table)
  
      ./parser <binary> segments
      
 View Elf sections (Section header table)
  
      ./parser <binary> sections
